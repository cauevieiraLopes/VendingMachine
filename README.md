# 🥤 VendingMachine-Cpp
Este repositório contém um simulador de máquina de bebidas desenvolvido em C++. O projeto foi criado como uma atividade prática de faculdade para exercitar a lógica de programação, o controle de fluxo e a interação com o usuário via console.

# 📌 Sobre o Projeto
O programa simula a experiência de compra em uma máquina de autoatendimento, permitindo que o usuário selecione um produto, escolha a forma de pagamento e valide a transação através de uma senha de segurança.

# 🚀 Funcionalidades
* Menu Interativo: Exibição de produtos com nomes e preços formatados.

* Validação de Dados: Uso de estruturas de repetição para garantir que o usuário escolha apenas opções válidas.

* Seleção de Pagamento: Suporte para simulação de transações via Cartão e Pix.

* Segurança Simbolizada: Fluxo de validação de senha para conclusão da venda.

# 🛠️ Conceitos de Programação Aplicados
Este código demonstra o uso prático de:

* Estruturas de Repetição: do-while para loops de validação de entrada.

* Estruturas Condicionais: switch-case para seleção de produtos e métodos de pagamento; if/else para validação de senhas.

* Entrada e Saída (I/O): Manipulação de dados via std::cin e std::cout.

# 💻 Como executar
1. Certifique-se de ter um compilador C++ instalado (como GCC/G++).

2. Clone o repositório ou copie o código para um arquivo ".cpp".

3. Compile o código: g++ vending_machine.cpp -o vending_machine

4. Execute o programa: ./vending_machine.
